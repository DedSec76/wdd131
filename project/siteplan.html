4. Color Schema

  • base-color: Main background of the website, the color that is seen in the background right now.
  • primary-color: Buttons, links, highlighted icons.
  • secondary-color: Titles, subtitles, borders, footers.
  • accent-color: Labels, button hover, or soft backgrounds.
  • neutro-white-color: Main text and contrasting backgrounds.
  • neutro-brown-color: Second color for Main text and contrasting backgrounds.
